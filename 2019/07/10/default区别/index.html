default区别

发表于 2019-07-10更新于 2019-07-10阅读次数

今天在给路由添加修改标题功能时，遇到了这个问题，开始时不清楚区别，导致一直报错，后来去查了一下才发现原来区别还挺大的。

export default的使用

export default向外暴露的成员可以使用任意的变量来接收

在一个模块中只能使用export default向外暴露一次

export的使用

在一个模块中可以同时使用 export default 和 export 暴露成员
使用 export 向外暴露的成员要使用 {} 接收，这种情况叫做：按需导出
并且一个模块中可以同时使用多个 export
