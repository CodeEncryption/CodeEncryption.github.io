Quasar创建PWA程序

发表于 2019-09-11更新于 2019-10-18阅读次数

近期工作的事情比较多，很久没有更新博客了（才不是因为懒呢）。最近迷上了摸鱼（一直摸鱼一直爽），喜欢看些热点新闻，经常去Mr.W的博客里的鱼塘（写到这里突然反应过来为什么叫鱼塘这名字，话说现在改名了但还是喜欢叫鱼塘23333）里摸鱼。抱着学习的态度去专门了解学习了下pwa方面的事情，并顺手自己模仿着搭了一个pwa玩玩,用了博主提供 的api。这里就说一下我查到所记得的东西，以及利用quasar创建pwa程序的过程吧。



PWA简介

现在说起pwa大部分人都不陌生了，但相信肯定很多人都想我一样之前只是听说过这么一个东西，而没有真正去实现，去完成一个pwa项目。

pwa可以算是小程序的前身了，为什么现在各种小程序这么火，pwa看起来优点更多却没有发展起来呢？

主要原因主要还是因为pwa程序在ios系统上的兼容性太差，许多增强体验的功能都被ios屏蔽了，导致苹果手机和安卓手机的体验差距巨大，而安卓手机也需要安装chrome浏览器（或者其他支持pwa的浏览器，不知道现在有哪些支持了），才能获得接近原生app的体验。

但是如果使用安卓手机又有chrome浏览器的话，pwa则成了一个非常棒的方式。

Quasar简介

Quasar（发音为/kweɪ.zɑɹ/)是MIT许可的开源框架（基于Vue），可帮助Web开发人员创建：
响应式网站
PWA（Progressive Web App）
通过Apache Cordova构建移动APP（Android，iOS，…）
多平台桌面应用程序（使用Electron）
Quasar允许开发人员编写一次代码，然后使用相同的代码库同时部署为网站、PWA、Mobile App和Electron App。使用最先进的CLI设计应用程序，并提供精心编写，速度非常快的Quasar Web组件。
当使用Quasar时，你不需要像Hammerjs，Momentjs或Bootstrap这样的额外重型库。它拥有这些功能，而且体积很小！

Quasar如同简介描述一样，只需要用Vue进行开发后，构建成不同模式即可。
有个小坑，quasar中文网的版本是旧版本(version<1.0)，和新版的语法不一样，建议看英文版的文档配合翻译使用

Quasar开发流程

想要构建PWA项目只能通过quasar-cli和vue-cli两种方式进行，我这边是通过推荐的quasar-cli进行创建的：

  1. 命令行运行

  2. 编写vue代码完成项目

  3. 运行quasar build -m pwa将程序打包成pwa

  4. 将构建好的程序放入服务器中即可

  5. 手机浏览器打开网站，添加到桌面后体验

注意：pwa程序不支持file协议，请一定要将程序放在服务器上跑而不是本地打开文件。

创建项目时根据自己的需要按照引导设置即可，不懂的设置直接回车就好。
